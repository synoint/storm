Basic usage

Firstly include the Syno Satisfaction Survey embed library in the <head> of your page

<html>
 <head>
  ...
  <script src="https://survey.synointcdn.com/embed_survey/v1/s-survey.js"></script>
  ...
 </head>
 <body>
 ...

Then place the following code into the body of the page to render survey invitation dialog

                            
                                
<div id="survey-container-id"></div>
...
